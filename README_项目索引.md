## 核心算法

```
probe() → 宽带噪音激发 → 256B响应
  ↓
scan()  → 35频段逐频扫描 → 与nls_baseline.json对比 → Δ向量
  ↓
balance() → Δ>0: 发射减弱信号(降噪)  
            Δ<0: 发射增强信号(激发)
  ↓
verify() → 复扫验证 → 若仍有偏差 → 循环
```

### 两种算法

| 算法 | 逻辑 | 六轮改善率 |
|------|------|:---:|
| **同频反相** | f_emit=f_detect, φ_emit=φ_detect+π | **76.0%** |
| **☀☽阴阳双频** | f_emit基于2ⁿ+Fibonacci权重 | 75.0% |
| 融合 | 两种算法联合 | 63.2% |
| 🌍舒曼锚 | 7.83Hz锚定 | 62.9% |
| 🔗原版(对照) | 不做反相 | 44.4% |
