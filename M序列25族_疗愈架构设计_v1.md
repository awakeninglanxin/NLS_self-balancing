## 五、25 族 → b9 映射表

b9 是唯一可调参数（b9→频率，b11/b15→振幅）。25 族的周期值需映射到 b9：

```
族周期 P → b9 = round(4 × log₂(P × 108 / 7.3728 / 3))

但这只在 P×108 落在 NLS 频段内时有效。
大多数族的低周期落在 b9 范围外（P=2→216Hz 远低于 NLS 最低频段 b9=1→26MHz）。

解决方案：做 M 次谐波上变频：
  f_therapy = P × 108 Hz × 2^M  (选择 M 使 f 落入 b9 有效范围)
  b9 = 4 × log₂(f / (7.3728 × 3))
```

## 七、预判任务 123

### 预判 1：多族框架会赢，但不是 25 族全上

**预测**：5-7 个核心族（Feigenbaum + Fibonacci + Sharkovsky + Prime + Lucas + FareyPhase）产生的治疗效果，会与 25 族全量持平甚至更好。原因是：
- 调参族（μ-count/φ(n)/熵/δ/Thue-Morse）的信息已被核心族的频率和振幅吸收
- 符号族（Kneading/Internal Address）的美感好但治疗效力待验证
- 跨语言族（连分数/KAM/Portrait）是数学上的存在证明，不是治疗必需

**验证方法**：A/B 对比：7 族 vs 25 族，看改善率差异是否 <5%

### 预判 2：CH1≠CH2 异构双通道会显著优于同频反相

**预测**：异构模式（模式1-5）的改善率预计比同频反相高 10-15%。
- 双频干涉（模式1）在神经系统治疗上会有明显优势——脑波差频是有神经科学支持的
- 混沌触发（模式2）对慢性病效果最好——Sharkovsky 混沌窗口的数学结构恰好匹配"松动僵化"的物理需求
- PCAP 分析显示原版 NLS 有 46% 的异频使用——说明原厂也有类似认知

**验证方法**：对同一器官在 A/B 模式下对比改善率

### 预判 3：最大的坑是振幅归一化

**预测**：25 族自然振幅差异极大（Feigenbaum A₄ = 0.002 vs Fibonacci A₄ = 0.18），不做归一化的话，Feigenbaum 的高频项会完全听不见，Sharkovsky 的混沌窗口会震耳欲聋。

**解决方案**：
```
A_normalized = (A_n / max_A_in_family) × A_target × Δ因子 × 五行系数
```
每族内部先归一化到 [0, 1]，再乘统一的发射振幅。

### 预判 4（新增）：协议 F 的 b9-1 近纯五度近似是足够的

**预测**：b9-1 → √2≈1.414 与纯五度 3:2=1.5 的 5.7% 偏差，在生物共振层面**不会产生可感知的失调**。理由：

1. 生物组织不是精密频率计——它响应的是一个频带，而非单一频点
2. 842 Hz 的 Solfeggio 体系本身就用近似而非精确频率
3. 纯律五度（3:2=1.5）和等程律五度（2^(7/12)=1.498）偏差 0.1%，但人类听觉完全无法分辨——5.7% 在生物共振层面同样在"容差带"内
4. 若追求精确纯五度，可以微调 b9 到非整数（b9=20.17→b9=20），但 NLS 手环只接受整数 b9

→ **用 b9-1 作为五度近似，不做额外纠正。** 三个整数间隔（Δ=1/2）已足够覆盖五行调谐需求。

```python
        b9_ch2 = upconvert_to_b9(p2)
        # 振幅 = 族衰减律 × Δ因子 × 五行系数
        amp1 = family_decay(p1, family1) * delta_factor(delta)
        amp2 = family_decay(p2, family2) * delta_factor(delta)
        # 发射
        send(b9_ch1, amp1, b9_ch2, amp2)
```
