## 2. 频谱分解

### 2.1 数列归属判定

对每个 b9 频段，计算其与两个数列的"距离"：

```
f(b9) = 7.3728 × 2^(b9/4) × 3

太阳距离: d_sun = min(|b9 - 2^n| for n=0..6)
太阴距离: d_moon = min(|b9 - fib(n)| for n=1..10)
```

### 2.2 权重分配

```
权重因子 = 1/(1 + 距离)
太阳权重 w_sun(b9) = 1/(1 + d_sun)
太阴权重 w_moon(b9) = 1/(1 + d_moon)
归一化: w_total = w_sun + w_moon
        w_sun' = w_sun / w_total
        w_moon' = w_moon / w_total
```

距离为 0（精确命中）→ 权重 100%。距离越大 → 权重越低。

## 3. 反相波生成算法

### 3.1 双频谐波模式（新）

```
对每个异常 b9 频段 (|Δ|>4):

  CH1 频率: b9_sun = b9 自身（如果太阳主导）
           或 b9_sun = 最近的 2^n (谐波牵引)
           
  CH1 振幅: b11 = 15 - sign(Δ) × |Δ| × w_sun' × 火行系数

  CH2 频率: b13_moon = 最近的 Fibonacci 数 (太阴牵引)
           使 CH1/CH2 频率比 = φ^n 或 Fibonacci 比
  
  CH2 振幅: b15 = 15 + sign(Δ) × |Δ| × w_moon' × 火行系数
```

### 3.2 举例

```
骨骼 b9=14, Δ=+12.0 (太阳主导, d_sun=2, d_moon=1)
  → w_sun=0.33, w_moon=0.50 → w_sun'=0.40, w_moon'=0.60
  → CH1: b9=16 (最近2^n), b11=15-12×0.4=10
  → CH2: b9=13 (最近Fibonacci), b15=15+12×0.6=22
  → CH1=297MHz 10% / CH2=250MHz 22%
  → 两频率比 = 297/250 = 1.188 ≈ 2^(1/4) 谐波

消化 b9=19, Δ=-8.0 (太阴主导, d_sun=3, d_moon=2)
  → w_sun=0.25, w_moon=0.33 → w_sun'=0.43, w_moon'=0.57
  → CH1: b9=16 (最近2^n), b11=15+8×0.43=18
  → CH2: b9=21 (最近Fibonacci), b15=15-8×0.57=10
  → CH1=297MHz 18% / CH2=500MHz 10%
  → 两频率比 = 500/297 = 1.684 ≈ φ (1.618) 黄金比例
```
