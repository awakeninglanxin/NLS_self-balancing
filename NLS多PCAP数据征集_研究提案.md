# NLS 多用户 PCAP 数据征集：从双人验证到人群统计

> **一句话**：我们有工具从任何人做 NLS 检测+疗愈时的 USB 抓包（PCAP）里提取疗愈指纹，现在缺的是更多的"人"。

---

## 1. 我们已经证明的事（2 人验证）

蔡**（女）和丽梦儿（女，邹某）两套独立数据——每人三份文件（检测 PCAP + 疗愈 PCAP + XCH 诊断报告），共 4,781 + 5,480 = 10,261 条疗愈 USB 命令：

```
┌──────────────┬──────────────┬────────┐
│              │   蔡**     │ 丽梦儿  │
├──────────────┼──────────────┼────────┤
│ 同频:异频    │   47% : 53%  │ 45:55  │
│ CH1 平均振幅 │     72.3     │  71.9  │
│ 对称反相率   │     10%      │   9%   │
│ CH2 降比     │    3.62×     │ 3.39×  │
│ 频段覆盖     │  b9=14 - 31  │ 14-31  │
│ 平均 burst   │    1.4 条    │  1.4   │
│ 器官排名     │ 肾>肺>甲>调>胰│肾>肺>甲>胰>调│
└──────────────┴──────────────┴────────┘
```

**结论**：算法协议骨架（同频比、对称率、频段范围、burst 长度）跨用户高度稳定，差异主要表现在器官排名（诊断个性）。这意味着我们逆推出来的原版协议是正确的——**不是一个人的偶然**。

## 2. 为什么 2 个人不够

2 个样本可以证明"这两份数据一致"，但回答不了：

| 需要回答的问题 | 最少样本量 |
|----------------|:--:|
| 同频比例 46±2% 是跨用户稳定还是巧合？ | 3-5 |
| 男女性别的振幅偏好是否有差异？ | 5-8 |
| 不同病种（甲状腺 vs 肝 vs 肾）的疗愈协议是否不同？ | 10+ |
| 能否从 PCAP 数据训练出一个"输入 scan delta → 输出最优协议"的模型？ | 20+ |
| 年龄、体质、慢性病对原版算法参数的调节效应？ | 50+ |

**2 个人的样本，方差无法计算，置信区间无限宽。** 任何 reviewer 都会问：你怎么知道这不是巧合？

## 4. 我们需要什么样的数据

极简——只需要你做一次 NLS 检测+疗愈时，在后台开着 USBPcap 抓包：

### 需要抓取的文件（3 个）

| 文件 | 用途 | 获取方式 |
|------|------|----------|
| **检测 PCAP** | 提取参考脉冲和检测探针的时序 | USBPcap 全程抓取检测过程 |
| **疗愈 PCAP** | 提取 12 维疗愈指纹（本次的核心） | USBPcap 全程抓取疗愈过程 |
| **XCH 报告** | 提取器官偏差值，和 PCAP 做双向验证 | MetaHunter 导出 |

### 同步记录（可选的，有就更强）

- 性别、年龄段
- 主要诊断结果（如"甲状腺结节""肝脂肪变"等）
- 是否首次检测还是复诊

### 操作步骤（5 分钟）

1. 打开 USBPcap，选择 NLS 手环的 USB 设备，点 Start
2. 正常做一次 NLS 全身检测（约 10-20 分钟）
3. 正常做一次 NLS 疗愈（和平时完全一样）
4. 点 Stop，保存 PCAP 文件
5. 导出 XCH 报告
6. 把 3 个文件发给我

**不需要你做任何额外操作。不影响正常检测流程。** USBPcap 在后台静默记录，文件大小约 20-50 MB。

## 5. 你的数据会被怎么用

```
你的 PCAP（匿名，仅标注性别+年龄段+诊断）
     │
     ▼
pcap_to_algo.py ──→ 提取 12 维指纹
     │
     ├── 加入人群统计池（和你无关的汇总统计）
     │
     └── 生成你的个人疗愈指纹卡
         如果你愿意，可以在后续自己使用网页版 NLS 时
         "加载你的卡"→ 原版算法变成你自己的版本
```

**数据不会包含你的姓名、电话、身份证号。** XCH 里的患者名建议在导出前匿名化（或者我来处理）。

## 6. 对贡献者的回报

1. **个人疗愈指纹卡**：你的 PCAP 会生成一份专属的 `algo_fingerprint_你的代号.py`，网页版 NLS 可以一键切换到你自己的原版算法。以后你自己用网页版做 NLS，`🔗原版` 按钮背后就是你自己的疗愈数据。

2. **匿名加入人群统计**：你的数据会成为"NLS 原版协议反向工程数据库"的一部分。当数据库达到一定规模，所有贡献者都能使用基于人群统计优化过的通用算法。

3. **优先看到对比报告**：我会为每位贡献者生成一份和已有数据对比的分析报告——你的疗愈指纹和别人有什么不同，可能反映了什么生理特征。
