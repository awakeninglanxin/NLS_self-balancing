# NLS 动态平衡仪 Android 手机版 — 使用说明书

> **版本**: v7.23 | **验证设备**: vivo10 (Android) | **更新日期**: 2026-07-05

---

## 一、简介

NLS 动态平衡仪是一款基于**频率共振反相原理**的经络平衡工具。通过 USB OTG 接口连接 NLS 手环设备，手机端以 18 个频段（20kHz ~ 7.37MHz）对人体经络进行扫描，检测能量偏差后自动发射反相频率实现动态平衡。

与电脑网页版功能完全对齐，支持 **7 种平衡算法轮转**、**实时三模式图表看板**、**传感器校准**、**七维对比统计**、**日志 TXT 导出**。

---

## 二、硬件要求

| 组件 | 要求 |
|------|------|
| 手机 | Android 8.0+（API 26+），支持 USB OTG |
| 转接线 | USB OTG 转接线（Type-C 或 Micro-USB） |
| 手环 | FTDI FT232 系列芯片，VID=0x0403，PID=0x6001 |

---

## 三、安装

1. 将 `nls_balance_v7.23.apk` 传输到手机
2. 打开文件管理器，点击 APK 安装
3. 如提示"未知来源"，前往 设置 → 安全 → 允许安装未知应用
4. 安装完成后桌面出现 "NLS动态平衡" 图标

---

## 四、操作流程

### 4.1 连接手环

```
① 将 USB OTG 转接线插入手机
② 将手环 USB 插头插入 OTG 转接线
③ 打开 APP，点击右上角「连接」
④ 弹出 USB 权限请求 → 点击「允许」
⑤ 状态栏显示 ● 手环已连接
```

> ⚠ **必须先连接，再校准。直接点校准会提示"请先连接手环"。**

### 4.2 传感器校准

```
① 确保手环传感器悬空（不要接触人体）
② 点击「📐 校准」
③ 等待 3~5 秒，日志显示 "✅ 校准完成: 18频段"
```

### 4.3 启动动态平衡

```
① 佩戴手环（传感器接触手腕或掌心）
② 点击「▶ 启动平衡」
③ APP 自动执行扫描 → 检测异常 → 反相治疗 → 复扫验证
④ 图表和日志实时更新
⑤ 每 7 轮输出一遍七维对比统计
⑥ 点击「⏹ 停止」结束
⑦ 点击「💾」导出完整日志 TXT
```

### 5.1 ☯ 五行雷达

五边形雷达图，五个轴（🔥火 🟡土 ⚪金 🔵水 🟢木）用各自五行颜色标注。绿色半透明区域为当前扫描偏差，偏离中心越远说明该五行能量越异常。

### 5.2 📈 频谱脊线

18 根柱状图对应 18 个器官频段。左侧高频（7.37MHz 松果体），右侧低频（20kHz 骨骼）。红色向上 = 能量偏盛，蓝色向下 = 能量不足。中间有 ~1.8MHz 分隔线。

### 5.3 ⚖ 七维对比

每 7 轮结束后自动统计，水平柱状图显示 7 种算法的改善率（↑改善/↓恶化）。按改善率从高到低排列，直观反应哪种算法对本人体质最有效。

## 六、七种平衡算法（与电脑网页版一致）

| # | 算法 | 名称 | 原理 |
|:--:|------|------|------|
| 1 | 🔗 原版 | Original | PCAP逆推，同频/异频比 54%/46%，振幅边界复刻 |
| 2 | 🔗 同频反相 | Legacy | CH1=CH2，偏高→压低振幅，偏低→提升振幅 |
| 3 | ☀☽ 双频 | Yin-Yang | CH1=2^n 数列，CH2=Fibonacci 数列，双通道协同 |
| 4 | ⚡ 融合 | Fusion | 反相地基 + 太阳/太阴数列微调（±2频段） |
| 5 | 🌍 舒曼锚 | Schumann | CH1 低频锚定（b9-6），CH2 偏差加权偏移 |
| 6 | 💧 水共振 | Water | 水分子团簇共振频率（2Hz~432Hz），频段索引映射 |
| 7 | ⚛ 幻数 | Jellium | 电子壳层幻数阶梯调幅（2,8,20…），奇偶分协议 |

APP 默认 **AB 模式**，每轮随机打乱 7 种算法顺序。每 7 轮为"一遍"，每遍结束输出七维对比报告。

## 七、日志信息解读

| 日志示例 | 含义 |
|------|------|
| `─── 第2遍: 融合 → ☀☽ → 舒曼 → …` | 当前遍的算法执行顺序 |
| `第5轮 — 扫描中… [🌍舒曼锚]` | 第 5 轮执行舒曼锚算法 |
| `⚡ 3项异常` | 本轮扫描发现 3 个器官能量异常 |
| `心脏/心血管 Δ=+8.5 1.84MHz` | 心脏频段能量偏高 8.5 点 |
| `✓ 全部频段平衡` | 本轮所有器官能量正常 |
| `════ 第1遍七维对比: 🔗原版 67% │…` | 每 7 轮统计各算法改善率 |
| `💾 已导出: NLS_Balance_...txt (XX条)` | 日志 TXT 导出确认 |

## 八、日志导出

1. 点击状态栏「💾」按钮
2. TXT 文件自动保存至 `Android/data/com.nls.selfbalancing/files/`
3. 文件名格式：`NLS_Balance_2026-07-05_12-30.txt`
4. 含完整的连接→校准→扫描→治疗→七维报告全部记录
5. 手机屏幕仅显示最近 50 条，导出包含从头到尾完整记录

## 九、常见问题

| 问题 | 解决方法 |
|------|------|
| 点击连接没反应 | 检查 OTG 线是否插紧，重新插拔手环 |
| 连接后灯不亮 | 手环需外接电源？检查 OTG 供电 |
| 📐校准提示"请先连接手环" | 必须先点「连接」成功后再校准 |
| 点击⚖对比闪退 | 需等第 7 轮完成后有统计数据再切换 |
| 图表不刷新 | 确认已启动平衡，检查日志是否有数据 |
| 五行标签看不清 | 已用五行配色（火红土黄金白水蓝木绿） |
| 之前版本闪退 | 已修复，请使用最新 v7.22+ 版本 |
