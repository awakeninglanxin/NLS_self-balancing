# NLS Dynamic Balancer

> 耳机的降噪算法 × 扩散模型 × NLS 手环 = 实时经络平衡仪

## 核心理念

```
扫描 → 检测Δ → 反相发射 → 复扫验证 → 循环
  ↑________________________________________↓
```

不再需要"诊断报告"——只要手环戴着，它就自动保持你的经络在平衡态。

## 工作原理

```
1. 宽带噪音激发 → 人体谐振 → 256B响应
2. 与空扫基线对比 → 计算Δ
3. Δ>0(偏高) → 发射减弱信号(降噪)
   Δ<0(偏低) → 发射增强信号(激发)
4. 复扫验证 → 继续循环
```
